Rule: Chapter တစ်ခန်းပြီးတိုင်း code ကို ကိုယ်တိုင် run ကြည့်ပါ။ ဖတ်ရုံနန့် မရပါ။

Course ပြီးဆုံးရင် သင်လုပ်နိုင်ဖို့အရာတိ

  • Java basic syntax ကို ကိုယ်တိုင် ရွီးနိုင်ပါဖို့
  • Input, condition, loop, method ကို သုံးပနာ small programs ဖန်တီးနိုင်ဖို့
  • Class/Object နန့် OOP foundation ကို အခြီခံကိုနားလည်ပါဖို့
  • Inheritance/Polymorphism, Collections, File I/O အတွက် bridge-level skill ရပါဖို့
  • Intermediate topics (DB, Spring) သို့ တက်နိုင်မည့် base ရပါဖို့

လေ့ကျင့်ရာမှာ မှတ်ထားရန်

  1. Copy-paste မပြုဘဲ code ကို ကိုယ်တိုင်ရွီးပါ
  2. Error message ကိုကြောက်မနိန်ဘဲ ဖတ်ပြီး ဖြေရှင်းလေ့ကျင့်ပါ
  3. Example ကို ပြင်ပြီး run ကြည့်ပါ (number/text values ပြောင်း)
  4. Chapter 14 mini project ၃ ခုကို မဖြစ်မနေ တစ်ခုမက ဆောင်ရွက်ပါ
